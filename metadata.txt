description=Performs accuracy assessment and validation analysis between two raster maps

# Detailed description (shown in plugin details)
about=	This plugin was developed with the assistance of DeepSeek AI and Claude AI (Anthropic).

	Raster Accuracy Assessment is a general-purpose QGIS plugin for performing 
    validation analysis between two raster maps (e.g. classified vs. reference). 
    It supports multiple sampling methods (random, stratified, systematic, CSV import), 
    flexible class mapping, and computes a comprehensive set of accuracy metrics including:
    Overall Accuracy, Cohen's Kappa, F1-Score (macro & weighted), Precision, Recall, 
    Producer's & User's Accuracy per class, Confusion Matrix, R², RMSE, MAE, and Bias.
    Results can be exported as TXT, JSON, or HTML reports. Compatible with QGIS 3.x.
    
    Desteklenen özellikler / Supported features:
    - Rastgele, katmanlı ve sistematik örnekleme
    - CSV dosyasından nokta yükleme
    - Sınıf eşleştirme arayüzü
    - Kapsamlı doğrulama metrikleri (OA, Kappa, F1, R², RMSE, MAE, Bias)
    - TXT, JSON ve HTML rapor çıktısı
    - Nokta katmanı olarak dışa aktarım (Shapefile)
